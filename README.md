# hormone-assay-data-analysis
A Python (Pandas &amp; Matplotlib) project analyzing hormone assay data, handling missing values, generating summary statistics, and visualizing results.
## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Steps
- Loaded and explored hormone assay data
- Handled missing values using column means
- Generated summary statistics (mean, min, max, std)
- Categorized testosterone levels into status groups
- Visualized hormone distributions and status counts
- Exported cleaned data and summary statistics

## Files
- `Hormone Assay Data Summary and Reporting.ipynb` – main analysis
- `hormone_cleaned.csv` – cleaned dataset
- `hormone_summary_stats.csv` – summary statistics output
